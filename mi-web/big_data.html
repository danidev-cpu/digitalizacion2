¿Qué es Big Data?

Big Data se refiere a conjuntos de datos tan grandes y complejos que precisan de aplicaciones informáticas no tradicionales de procesamiento de datos para tratarlos adecuadamente. El uso de big data ayuda a las empresas a tomar decisiones más informadas y a descubrir patrones ocultos.
